Why Scaforge?

Unlike monolithic boilerplates, Scaforge lets you build exactly what you need.

Minimal by Default

Start with just the framework and essentials. No bloat, no unused code.

Plugin Everything

Add any feature as a plugin. Auth, database, payments - all optional.

Auto-Integration

Plugins automatically wire up with each other. Add auth + API = protected routes.

Framework Agnostic

Same plugin system works across Next.js, Nuxt, TanStack, and Hydrogen.

Get Started in Seconds

One command to create your project. Add plugins as you build.

1

Create your project

$ npx create-scaforge my-app
2

Add plugins you need

$ npx scaforge add api-trpc # Type-safe API

$ npx scaforge add auth-clerk # Authentication

$ npx scaforge add db-prisma # Database ORM

3

Start building

$ pnpm dev
